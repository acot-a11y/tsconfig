# tsconfig

> Shared TypeScript config for acot projects.

## Installation

```bash
$ npm install --save-dev @acot/tsconfig
```

## Usage

Write the following code in your `tsconfig.json`.

```json
{
  "extends": "@acot/tsconfig",
  "compilerOptions": {
    "outDir": "lib",
    "rootDir": "src"
  }
}
```
